# 025 — 转写语言管理优化

## 概述

优化 Apple Speech 转写语言的管理体验，使设置页、首页和视频转写页的语言选择更加直观合理。语言下拉选只显示已安装的转写语言，完整的语言列表移至独立的"管理语言"弹窗中

## 背景

- Apple 的 `AssetInventory.reserve(locale:)` API 限制每个 app 最多保留 5 个 locale 预留槽位
- 已下载的语言数据作为系统共享资源会持久保存，释放预留不会删除已下载的数据
- 当前设置页直接展示全部支持的语言列表，用户可能不清楚哪些语言已经可用

## 需求

### 1. 设置页

- 只显示已安装（installed）的转写语言列表，可以移除语言（卸载不实际删除）
- 顶部提示信息说明只能同时激活 5 个语言
- 底部添加"刷新"和"管理语言"按钮
- "管理语言"打开弹窗（Sheet），包含完整的语言列表
- 弹窗中可以选择/移除语言，完成后自动刷新内外列表，底部为"完成"按钮

### 2. 首页转写语言下拉选

- 只显示已安装的转写语言
- 如果用户没有下载过任何转写语言，显示提示或警告
- 在下拉选最后添加"管理语言…"选项，点击跳转到设置页

### 3. 视频转写语言下拉选

- 与首页逻辑一致：只显示已安装语言
- 无下载语言时显示"无"
- 底部添加"管理语言…"选项跳转设置页

### 4. 注意事项

- 不影响翻译功能的使用
